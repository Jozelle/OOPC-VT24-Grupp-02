# OOPC-VT24-Grupp-02

Ett bokningssystem för en bilverkstad, byggt i C# med WPF. Lösningen är uppdelad i flera projekt: presentationslager (WPF/MVVM), affärslager, datalager och entiteter.

## Kör projektet

Krav: .NET SDK samt Visual Studio 2022 (eller motsvarande) på Windows, eftersom presentationslagret använder WPF.

1. Klona repot.
2. Öppna `OOPC-VT24-Grupp-02.sln` i Visual Studio.
3. Sätt `CarService.PresentationLayer.WPF` (eller `CarService.PresentationLayer.WPF.MVVM`) som startprojekt.
4. Bygg och kör med F5.

## Om kursen

Projektet är en gruppuppgift i kursen *Objektorienterad programmering i C#* vid Högskolan i Borås.
